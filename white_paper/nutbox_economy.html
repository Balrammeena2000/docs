# Nutbox Economy

# 总体目标

Nutbox的第一目标是为PoS链的Staking Token提供流动性，除此之外，更为重要的是支持各PoS链上的生态社区发行cToken，利用“Staking + DeFi”完成社区的启动。我们认为，只有PoS链上社区丰富才能带来PoS链上的安全和价值。

Nutbox作为一个Polkadot parachain通过Polkadot与其他链相连，与其他parachain共享基本共识，确保安全性。故Nutbox会预留一定比例的Token作为支付Parachain插槽或线程的成本。

# Nutbox Token

WNUT是一种本机的实用性代币，旨在推动Nutbox生态中各项协议的发展。

WNUT亦是社区成员之间交易的媒介，引入其的目的是在Nutbox生态内，提供一种方便参与者互动、安全支付和结算的模式。特别要注意的是，WNUT只能在Nutbox生态上使用，除了作为生态内部使用及交易的权利外，没有其他明确或暗示的权利。

# 发行

WNUT的初始总量为10亿，代币将在Nutbox Network主网启动后进行分发。主网启动前，Nutbox v1网络将通过ERC20、TRC20和Substrate测试网活动进行PNUT的分发。

WNUT采取半衰期的方式按照区块生产，其分发分为三部分：1、Yield Farming（70%）；2、Nutbox.dao Fund（20%）；3、Nutbox验证者（10%）。

# Yield Farming

Nutbox最重要的是，推动Staking coin offering、Liquidity Staking Token、Nutbox DApp laucher的发展，将tToken引入各个社区的应用场景之中。

Nutbox将考虑对支持Staking coin offering和Liquidity Staking Token的通道进行激励，只要开发人员将Peanut质押挖矿和Nutbox质押资产流动性协议集成到其钱包、DApp或Exchange中，都将获得一定比例的奖励（设置benefit ratio）。

Nutbox亦将激励为tToken、WNUT提供流动性的行为：1）tToken、WNUT相关交易对；2）tToken相关抵押借贷池；3）Token转化为tToken。各部分分发WNUT的比例，将根据各池创造的收益进行调整。参与Yield Farming的交易对或借贷池名单，将由WNUT持有者投票表决，名单每6个月更新一次。

此部分WNUT将占据总量的70%，其中的一部分将作为整个生态系统的激励措施，包括PLO插槽拍卖、Oracle使用和跨链相关的成本。

# 国库分发

Nutbox在进行Yield Farming的同时，同步生成20% WNUT进入国库（Nutbox.dao Fund）。这部分WNUT将用于激励整个生态系统的建设，包括 Nutbox创始团队、市场与运营、Nutbox v1 Steem and Tron Cross-chain Staking network PNUT持有者、私募融资以及其他参与Nutbox生态建设的成员。

作为Nutbox的v1网络，Steem and Tron Cross-chain Staking network是探索SCO、Liquidity Staking Token、Yield Farming、Community DAO、DApp launcher的核心。其相对于Nutbox Blockchain，一如Kusama之于Polkadot。Nutbox将把国库的大部分WNUT（12%）分发给私募融资及v0.1 PNUT持有者。

# Nutbox验证者

Nutbox未来作为一条独立的Parachain，在与Polkadot共享共识安全的同时，也有着自己的验证者。总量的10% WNUT将在Nutbox blockchain上线后，对Nutbox验证者进行持续奖励。

# WNUT价值捕获

# 一种价值储存物

价值储存物指的是能够保值的资产，不会随时间流逝而大幅贬值的资产。流通中每个WNUT的价值都由实际的价值支撑，所有WNUT交易都在区块链上公开可见，其设计使其在波动巨大的市场中也能保持价值稳定。

# 一种交换媒介 / an exchange intermedia

交换媒介是指一切可代表价值标准，被用来促进商品或服务出售、购买或交换/交易的事物。WNUT作为一种交换媒介可通过Nutbox跨链接入各PoS链生态，与对应场景匹配，使用WNUT来达成交易，为WNUT增值。比如，在Steem区块链可支付WNUT获得nutbox.mine点赞。

# 一种服务费 / a kind of service fee

由Nutbox协议构成的去中心化自动网络24×7全天候地为成员提供服务。不管社区项目团队通过Nutbox协议发行cToken、启动Yield Farming、配置DApp launcher，还是社区成员生成tToken，都需要支付一定的WNUT来获得网络增值服务。

# 一种系统参与权 / a participant right

参与备选的验证者节点需要质押一部分WNUT，才能被允许加入Nutbox验证集，此亦会增加验证者Slash的成本。抵押WNUT的多少根据验证者节点的性能得分进行增加或减少。

WNUT亦可用于链上投票、技术委员会、国库分配、公投治理、节点选举等链上治理功能，持有WNUT可以在Nutbox网络治理中获得权利。
